**Chapter One 产品** 

这是一个无病呻吟的文章，如果不想给自己带来负能量请赶紧关闭。

最近做了一个基于 QT 的小项目 AI.xyz。xyz 指万物，这个工具可以实现各种 AI 功能。文本对话、语音合成、图片生成等等。

说来也搞笑，最开始只是我想换个新键盘，但是刚买了电脑，媳妇不让。我琢磨怎么能挣点外快，我又啥也不会，想了半天最后看看卖代码行不行。题材选了今年一直很火的大模型，为啥做这个。我的思维也想不到什么好的项目，基于Qt开发的什么笔记软件、各种播放器、聊天软件、画板、翻译软件啥的都太多了，基于Qt实现调用大模型对话的我到没看到过。

为啥没有也挺简单的，都2024年了，这类纯调API的桌面客户端软件没必要用Qt，事倍功办了属于是。各家都有sdk，python一行代码就搞定了，正经做项目的不会用Qt来搞这个，没人做刚好我做。当时给自己想的目标群体就是大学生的毕设或者课程设计，用Qt实现个大模型对话，当个作业多好。自己预期：业余时间一个月写完，能卖个20份，6000块换个键盘鼠标耳机啥的，这事就结束了。

事实证明我太高估自己了，就只卖出去一份。期间还被喷过好几次，最多的观点就是，作为程序员花300买你这么个demo，还得自己看，自己改还不如重写。我都没反驳，批评我就听着，因为就是这样，反过来如果我工作需要做这么个东西，我也不会花300去看别人代码，也不知道他水平，看下来最后自己可能还要重写。问的大多数人一开始就不是我的目标用户。做之前我就知道，有点工作经验的应付下就行，愿意怼就怼，不会买的，换位思考因为我自己就不会买。让我郁闷的就是，咋没有学生买来当作业啊。现在想想，本身产品就不行，更何况那俩月尼玛是暑假。

xyz 最开始想的是各种语言 API 。虽然没人要，也没人用，我自己都不用，API实现对话的效果跟直接使用网页版差距太大了。奈何自己做的过程感觉很爽，还是写代码好玩，就琢磨不行堆功能，功能多了虽然没人买但或许就有人用了。

正好当时周末做视频需要文本转语音，平时很少去找这方面工具，每次转换我都头大，要各种找，各种试听，网站不是付费就是不好用。干脆我自己先加语音合成吧。原来只是百度随便搜，后来正经去调研的时候，发现了很多免费好用的文本转语音，要是一开始排在前边的不是那些打广告的垃圾工具我应该不会加语音合成，直接研究怎么调用图片生成的API了。

当我把语音合成做完后，一切都不一样了。因为 AI.xyz 终于有用户了，我自己。到现在我一直用了两个月了，每期视频都是用他生成的配音。对于我自己这个低频需求而言，价格免费，页面简洁顺眼，足够好用了。当然需要有一定的门槛，虽然免费但需要有自己的Azure账户。我自己也知道，有账户的人群也不会看得上我这个工具的。反正我自己在用，觉得好用，每次用都给了我大量的正反馈，这就够了。

后来继承了几个云的图片生成、涂抹消除、扩图、涂抹编辑。我知道这玩意没人用，因为我自己就不用。有动手能力搞自己API的，如果要用也是去各种平台，比如哩布哩布。

后来增加了 Ollama 本地部署，和ComfyUI的本地部署。Ollama除了开发的时候，我也不是怎么用。ComfyUI好像本身就要发布客户端了，集成进 AI.xyz 后我自己没事就画生成几张，成就感满满，算是实现出图自由吧，比掉豆包和阿里的 API 稳定多了。

最近国产对话的大模型最近又加了三个，其实感觉他们效果都差不多，加上后除了让我自己自恋一下，看我写的架构多好，30分钟就能加一个新的大模型，也没啥用。

**Chapter Two 放弃** 

这周把咸鱼、工坊和原来说卖源码的评论都删除了，刚开通的激励和充电也都关了。哔哩哔哩手机版我也卸了，上传和看数据啥的就用网页版吧，省的自己没事就一直刷。

愿意有很多，压死骆驼的最后一根稻草就是提现。把源码挂工坊后，用媳妇手机下了一单，用了一周才体现。原因就是绑定不了银行卡，联系客服，换手机、换网络、重新安装、反复登录都没成功。最后到技术手里，结论：安卓手机不能提现，必须苹果手机。换成苹果后果然直接提出来了。有BUG有故障啥的，我是程序员，我也都理解，也没觉得这事有啥问题，出问题平台改就是了。

只是心里一直有点疑惑，借着这事问下自己，我这是再图个啥。源码定价300，快四个月就卖了一份。就算买他个20份还没我这个月税交的多，何必呢。坚持做下来是因为看着他一点点完善，即使没用户心里也很爽，写代码本身让我幸福。天天琢磨怎么卖出去，都变得不快乐了。也许离成功就差一点，再坚持几个月就卖爆了。没那个命，放弃了，不卖了。

不卖了直接开源？又有点不甘心，以后再说吧。对，我就是B子立牌坊，又当又立，即要又要的那种人。项目接着做接着玩，长时间坚持一件事情确实能从中收获不少快乐，要是适当的来点正反馈就更好了，比如点赞、评论个加油啥的。

**Chapter Three 回顾** 

依稀接的第一次购买付费内容是在18年，当时刚毕业。在咸鱼150块找人写一个打开STL模型的功能。付款的时候先给了100，看了给我的源码感觉有些不值，就是安装QTCreator后带的那个Helo GL Demo，但是模型换成本地STL。剩下的50拖了一周没给，被拉黑骂了一顿，我也没还嘴。那会还年轻，担心被找上门或者网上曝光啥的，被拉黑后转给对方100，一共给了200，那年在杭州，我工资税前四千五。确实应该被骂，没有契约精神，而且是我先找的他，人家也确实付出了时间。

在20年第一次接到了外包，对方报价5万。从接到任务到拿到全款我做了一个月。那个月真是每天巨兴奋，一度飘起来了，觉得自己巨牛逼，甚至想辞职自己做独立开发创业。当时我的工资一个月才18k，那个项目如果要我报价，我就要5000，给3000就能很开心的做完。结果比预期翻了10倍。

现实很残酷，第二次接到外包间隔了3个月，在一个QT交流群里给一个学生做Demo，源码发了远程调试好后被拉黑了。报价要的300，一分没收到。本来打算群里@出来骂一骂啥的，想了想我第一次不给别人尾款后紧张的小心情就算了，没吐槽也没骂人。打哪以后我就基本不再看什么交流群了，慢慢也都退了。

虽然很喜欢编程，但是工作几年以后才认识到自己就是个拉跨的初级程序员。刚转换那两年喜欢在CSDN上写博客，后来翻翻看自己之前写的，感觉自己就是个拙略的搬运者。初心写博客记录自己学习过程是好的，但自己写的很多笔记更多的是误人子弟。也就不再CSDN写什么学习笔记了。

当然，无论什么工作，记录、整理、积累都是很有必要的，选一个舒适的地方写就完了，觉得那个平台或者软件舒服就在那里，没有什么优略好坏。CSDN是一个很好的快速入门平台。不过自从GPT火了之后，对于想要快速入门，结合GPT应该有更好的体验。

没有尝试过知识付费，自己水平有限，做也只能搞搞针对初学者的课程。越发感觉针对初学者的课程以前还能赚到一些钱，现在有了ChatGPT，初学者课程生意对于个人而言已经死掉了。AI可以很好的回答所有的新手问题，而且回答的比大多数人好。AI不能提供的是那些来之不易的深刻体悟和个人见解，那些很难包装成商品售卖，更适合在实践中指点。

95年的，除了中彩票还是打工才靠谱，体验式幸福我应该是无缘了，还是寻找反思式幸福吧。持续做一件事情确实能给自己带来快乐。业余开发这几个月，虽然没有用户，

**Chapter Four 再出发** 

最近特别想搞个pocket来拍vlog学剪辑，很羡慕那些数字游民和裸辞旅居的博主。又怕自己脑子一热买来就没有下文了，还是要冷静。

应该很少有人会看到这里了，立个Flag，把这篇博客做成文字视频。现在B站每个视频播放大概200，如果这个视频播放能到3500我就买基础款，能到4500就买全能套装。当然大概率还是200，那我就删了视频，反正也没人知道。

抛开盈利，记录生活。

虽然写的很丧，其实我感觉自己是一个普通且谜之自恋的人，每天还是很积极和正能量的。

无论你处于人生的什么阶段，无论是否看到这，都祝愿你充满能量且幸福。
